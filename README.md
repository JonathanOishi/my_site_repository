# Jonathan Oishi Portfolio

Portfólio pessoal desenvolvido em Flutter, projetado para apresentar minha trajetória profissional, experiências, competências técnicas e projetos de forma moderna, responsiva e escalável.

O objetivo deste projeto foi construir uma aplicação web com foco em performance, organização arquitetural e experiência do usuário, utilizando boas práticas de desenvolvimento e uma estrutura preparada para futuras evoluções.

## Sobre o Projeto

Este portfólio foi desenvolvido para centralizar minha apresentação profissional em uma única plataforma, permitindo que recrutadores, empresas e parceiros conheçam minha experiência, formação e principais projetos.

A aplicação foi construída utilizando Flutter Web e segue uma arquitetura modular baseada em componentes reutilizáveis, facilitando manutenção, expansão e escalabilidade.

## Principais Seções

* Apresentação inicial (Hero Section)
* Minha trajetória profissional
* Stack tecnológica
* Experiência profissional
* Formação acadêmica
* Projetos em destaque
* Contato
* Rodapé institucional

Além disso, o projeto conta com navegação responsiva adaptada para desktop, tablet e dispositivos móveis.

## Tecnologias Utilizadas

* Flutter
* Dart
* Material Design 3

## Arquitetura

O projeto foi estruturado seguindo princípios de organização e separação de responsabilidades.

```text
lib/
├── core/
│   ├── responsive/
│   └── theme/
│
├── features/
│   └── view/
│       ├── navigation/
│       ├── pages/
│       ├── sections/
│       └── widgets/
```

### Destaques da Arquitetura

* Componentização de widgets
* Estrutura baseada em features
* Responsividade centralizada
* Design System unificado
* Fácil manutenção e expansão
* Separação clara entre regras visuais e lógica de apresentação

## Design System

Todo o sistema visual foi centralizado para garantir consistência em toda a aplicação.

### AppColors

Todas as cores do projeto são gerenciadas através da classe `AppColors`, eliminando valores hardcoded e facilitando futuras alterações de identidade visual.

### Responsividade

A adaptação entre mobile, tablet e desktop é realizada através de utilitários centralizados, garantindo uma experiência consistente em diferentes tamanhos de tela.
